Detailed Description

This simple adapter implements a generic transport message providing listener based on a DoIP specific message providing listener.

Whenever a specialized DoIP specific transport message provider is required there will still be some transport layers that only accept a generic transport message provider. This adapter implements such a generic transport message provider by mapping all calls to the more specific DoIP transport message provider.
