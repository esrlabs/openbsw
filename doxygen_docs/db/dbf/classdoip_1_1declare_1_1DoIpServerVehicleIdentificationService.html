class doip::declare::DoIpServerVehicleIdentificationService< DatagramSocket, NUM_UNICAST, NUM_SOCKETS, NUM_REQUESTS, NUM_ANNOUNCEMENTS >

Implementation of a DoIP vehicle identification server that can work on a given number of UDP sockets that can be configured separately.
