Detailed Description

template<class ServerSocket, class Socket, size_t NUM_SERVER_SOCKETS, size_t NUM_PLAIN_SOCKETS, size_t NUM_TLS_SOCKETS = 0U>
class doip::declare::DoIpServerSocketHandler< ServerSocket, Socket, NUM_SERVER_SOCKETS, NUM_PLAIN_SOCKETS, NUM_TLS_SOCKETS >

Socket handler class that allocates both TCP server sockets and connection sockets.

Template Parameters
ServerSocketserver socket class
Socketsocket class
NUM_SERVER_SOCKETSnumber of server sockets to allocate
NUM_PLAIN_SOCKETSnumber of connection sockets to allocate. One of these is required for each DoIP connection (non-TLS and TLS) and thus limits the maximum number of DoIP connections independent of the connection type.
NUM_TLS_SOCKETSnumber of connection sockets (TLS only) to allocate. This limits the maximum number of TLS connections.
