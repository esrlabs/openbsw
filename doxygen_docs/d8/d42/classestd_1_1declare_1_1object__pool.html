Detailed Description

template<class T, std::size_t N>
class estd::declare::object_pool< T, N >

The actual implementation of the object_pool class. Use this to declare your object_pool, but use the base class in your API. [TPARAMS_BEGIN:declare_object_pool]

Template Parameters
TType of elements of this object_pool.
