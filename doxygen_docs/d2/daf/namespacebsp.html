Detailed Description

Contains abstract base class every class which wants to get jobs executed in a background task has to implement.

Contains interface of a class which runs jobs for other components during a background task (idle task).

Note
* In order to influence the system as little as possible the background task must be preemptive and jobs executed should have as little critical sections as possible if any.
