bool ethernet::TxBuffers::writeBuffer ( uint8_t  txDescriptorIndex,
::etl::span< uint8_t const >  data,
bool  lastBufferInFrame 
)

Write buffer (descriptor), neither copy data nor send. To send the frame, call commitFrame() after one or more calls to writeBuffer().

Synchronization

The call(s) to writeBuffer() and commitFrame() must not be interrupted; so protect them with e.g. ScopedCriticalSection.

Parameters
txDescriptorIndexIndex of the buffer descriptor to write to.
lengthTransmit data size.
pDataPointer to transmit data.
lastBufferInFrameSet to true on the last buffer in the frame.
Returns
false in case of error, true on success.
