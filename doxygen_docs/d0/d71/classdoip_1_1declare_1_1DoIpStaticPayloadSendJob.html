Detailed Description

template<size_t BufferSize>
class doip::declare::DoIpStaticPayloadSendJob< BufferSize >

Class that can be used to represent a DoIP send job with preallocated payload buffer.

Template Parameters
BufferSizesize of the allocated buffer i.e. the maximum size of a payload that can be stored within this send job.
