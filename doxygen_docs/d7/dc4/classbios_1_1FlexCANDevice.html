◆ getTransmitBuffer()

uint8_t bios::FlexCANDevice::getTransmitBuffer ( can::CANFrame const &  frame,
bool  callbackRequested 
)

Searches for a free CAN Buffer for a given CAN frame id. if there is already a frame with the same id in one of the buffers, a free buffer with a buffer number greater than the one already in the hardware queue is returned else the lowest empty buffer is returned.

Parameters
frameCAN frame to be queued (id and isExtended is used only).
callbackRequestedIf true, CALLBACK_TRANSMIT_BUFFER (if free) is returned.
Returns
Free CAN buffer number for the id
  • TRANSMIT_BUFFER_UNAVAILABLE if there is no buffer available
  • buffer number if a free buffer is available
