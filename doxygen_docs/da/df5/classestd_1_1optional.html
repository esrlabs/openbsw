Detailed Description

template<typename T>
class estd::optional< T >

A wrapper representing a value that may or not be valid. The instance of type T is stored in memory residing inside the optional<T> object itself.

Constructor & Destructor Documentation

The copy constructor copies both the contained object (if there is one), as well as the validity state. Note that this uses the copy constructor of T and thus only works for copyable types.

◆ value_or()

template<typename T >
T estd::optional< T >::value_or ( T const &  fallback) const
inline

Shortcut to default to a fallback value. The return value will either be a copy of the contained object (if there is one), or a copy the provided fallback parameter. Note that the result is returned as a copy. This in unavoidable because most of the time a temporary object will be passed as fallback parameter a reference to it would be dangling after the function returns. Therefore use this method should only be used for simple types.
