Detailed Description

template<size_t NUM_SOCKET_GROUPS>
class doip::declare::DoIpServerTransportLayer< NUM_SOCKET_GROUPS >

Class implementing a transport layer on a given bus supporting a number of socket groups. Socket groups are used to create partitions of connections that are independent regarding alive checks

Template Parameters
NUM_SOCKET_GROUPSnumber of socket groups to support
